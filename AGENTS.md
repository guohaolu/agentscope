# AGENTS

## 项目定位

本项目主要用于学习 Python 知识、AgentScope 框架的使用方式，以及相关设计思路。

## 文档记录规则

1. 生成的源码分析、语法学习、框架理解与设计思路总结，都需要以 Writerside 的方式记录下来。
2. 新增或修改文档时，优先放在 `Writerside/` 目录下，并遵循该目录现有的组织方式。

## 语言规则

1. 所有注释必须使用中文。
2. 所有文章和说明文档必须使用中文。

## Writerside 规则

1. Writerside 中 Markdown 文件的 H1 标题必须与文件名一致。
2. 如果文件名发生变化，必须同步修改 H1 标题，保持两者完全一致。
